# Vrit Technologies Frontend Trainee Tasks

## Used Tech Stack

- Next.js (React Framework + Router)
- TailwindCSS (CSS Framework)
- shadcn ui (styled components)
- framer motion (Animations and transitions)
- React Query (API handling & server state management)
- Zustand (client side state maagement for posts )
- React Hook Form + zod (Form Handling and validation)

## Main Directories

- `/hooks` -> custom hooks
- `/app/(posts-dashboard)` -> post related routes & components
- `/app/(ui-tasks)` -> ui tasks routes and components
- `/components` -> global components
- `/types` -> types for posts and users
- `/validators` -> validator schemas
- `/store` -> zustand store

Note: Local components are stored inside nearset shared directory at `_components`
